# Шифровальщик Python #
Программа способна *зашифровывать* и *расшифровывать* слова в трёх системах:
- **Азбука Морзе**
- **Шифр Цезаря**, где пользователь сам задаёт сдвиг по алфавиту
- **Табличное шифрование**, где каждая буква обладает определённым кодом, в зависимости от строки и столбца

Программа работает со словами **русского** и **английского** языков.
